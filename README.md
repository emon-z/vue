# vue
项目描述：本项目是一款移动端项目，采用vue框架构建，其中涉及swiper触控滑动模块，slide子页之间采用了懒加载技术保证用户体验，iscroll上拉加载下拉刷新模块，
购物车模块与登录注册模块均采用了本地存储技术。
